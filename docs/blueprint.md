# **App Name**: Baobab Golf

## Core Features:

- Landing Page: Landing page with clear navigation to register teams, access scorekeeper dashboard, and spectator dashboard.
- Team Registration: Team registration form to collect team details, with validation and submission.
- Scorekeeper Dashboard: Scorekeeper dashboard to manage and start games, view team overviews, access leaderboards, and use chat functionality.
- New Game Setup: Start a new game by selecting either a 9-hole or 18-hole option.
- Live Scoring: Live score input interface with hole-by-hole scoring for four players and automatic best 2 score selection.
- Spectator Dashboard: Spectator dashboard to view game summaries, leaderboards, and track live scores in real-time, alongside a public chat.
- PWA Support: PWA configuration for install prompt and manifest setup to ensure the application is installable on devices.

## Style Guidelines:

- Primary color: Accent red (#e4002b) for interactive elements and highlights, providing a strong visual cue in the dark interface.
- Background color: Dark grey (#121212) for the main background to create a modern and sophisticated dark mode theme.
- Card/container colors: #1e1e1e, #2e2e2e.
- CTA red (accent): #FF405A.
- Text: #ffffff, #a6a6a6 for descriptions
- Font: 'Inter' (sans-serif) for all text elements (headlines, body, labels, etc.). Note: currently only Google Fonts are supported.
- Cards with transparent backgrounds and blurred backdrop effect to display data, allowing the background image to subtly show through.
- Fixed bottom navigation bar in both Scorekeeper and Spectator dashboards for easy access to main sections.
- Rotating golf ball animation on the landing page to draw user attention and create a dynamic effect.
